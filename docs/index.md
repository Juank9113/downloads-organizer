# 📁 Downloads Organizer

![Tests](https://github.com/Juank9113/downloads-organizer/actions/workflows/tests.yml/badge.svg)
![Python](https://img.shields.io/badge/python-3.8%2B-blue)
![License](https://img.shields.io/badge/license-MIT-green)

**Organiza tu carpeta de Descargas automáticamente en Linux, macOS y Windows.**

## 🚀 Características

- 🗂️ **Organización automática** por tipo de archivo
- ⚙️ **Configuraciones personalizables** mediante JSON
- 🧪 **Modo simulación** para previsualizar cambios
- 📊 **Estadísticas detalladas** de archivos organizados
- 🐧 **Multiplataforma**: Linux, macOS y Windows
- 🔔 **Notificaciones** de escritorio opcionales
- 📈 **Reportes semanales** automáticos

## ⚡ Inicio rápido

    git clone https://github.com/Juank9113/downloads-organizer.git
    cd downloads-organizer
    python3 src/organizer.py --simulate
