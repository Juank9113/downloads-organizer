# Ejemplos de Configuración - Downloads Organizer

Este directorio contiene ejemplos de configuración para personalizar el comportamiento del organizador de descargas.

## 📁 Archivos Disponibles

| Archivo | Descripción |
|---------|-------------|
| `custom_rules.json` | Configuración completa con todas las categorías predefinidas |
| `README.md` | Este archivo, documentación de ejemplos |

## 🚀 Cómo Usar los Ejemplos

### 1. Copiar el ejemplo como base

```bash
# Copiar la configuración de ejemplo
cp examples/custom_rules.json mi_configuracion.json

# Editar según necesidades
nano mi_configuracion.json
# o
code mi_configuracion.json
