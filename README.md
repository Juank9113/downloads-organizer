## 🚀 Características

* 🗂️ **Organización automática:** Clasifica archivos por tipo (imágenes, documentos, videos, etc.).
* ⚙️ **Personalizable:** Configuración flexible mediante archivos JSON.
* 🧪 **Modo simulación:** Previsualiza cambios sin mover archivos reales.
* 📊 **Estadísticas:** Reportes detallados de archivos organizados y espacio liberado.
* 🔔 **Notificaciones:** Alertas de escritorio y sonido (opcional).
* 🐧 **Cross-platform:** Compatible con Linux, Windows y macOS.

## 📋 Requisitos

* Python 3.7 o superior.
* No requiere dependencias externas para las funciones básicas (usa librerías estándar).
* Para funciones avanzadas, revisa `requirements.txt`.
